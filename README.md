# massive-json-generator
generator of thousands fake data in json format

---

### 1. Requirements
```
node --version
v10.15.3
```
```
npm --version
6.9.0
```

### 2. Installing

1. Clone the repository
2. Install dependencies:

```
npm install
```

### 3. How to use?
Run on terminal:

```
node generator -n 100 fakeData.json
```
